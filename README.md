功能說明
<ul>
    <li>網址字元數最高1000</li>
    <li>自訂錯誤訊息內容，並加入檢查 必填、必須為合法網址</li>
    <li>已經縮過的網址會將記錄翻出來，不會另外生成新的縮網址</li>
</ul>

建置環境
<ul>
    <li>linode VPS</li>
    <li>ubuntu 20 LTS</li>
    <li>laravel 8.x</li>
    <li>Docker version 20.10.12</li>
    <li>使用laradock部署環境</li>
</ul>
